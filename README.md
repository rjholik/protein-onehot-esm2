# protein-onehot-esm2

A collection of notebooks for protein sequence encoding, embeddings, and 3D structure visualization using modern Python tools.

## Overview

This repository contains hands-on notebooks exploring computational protein representations, including:

* One-hot encoding of amino acid sequences
* PyTorch tensor-based sequence processing
* ESM2 protein language model embeddings
* Heatmap visualization of encoded proteins
* Interactive 3D protein structure rendering with py3Dmol
* Bioinformatics and machine learning workflows

This repository is a work in progress and will continue to expand with new notebooks, experiments, and methods over time.

## Included Notebooks

### Sequence Encoding

* Amino acid indexing
* One-hot encoding
* Tensor preparation for deep learning

### Protein Language Models

* ESM2 embeddings
* Learned sequence representations
* Feature extraction workflows

### Structure Visualization

* Interactive 3D protein rendering
* PDB structure retrieval
* Scale bars and annotation labels
* Molecular exploration with py3Dmol
